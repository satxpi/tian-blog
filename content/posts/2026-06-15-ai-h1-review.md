六月过半，该写个半年度回顾了。

2025年是"爆发年"，每个月都有炸裂新闻。2026年前六个月，感觉不一样——不再是砸场子式的发布，而是**各走各的路**。

## 分化开始了

去年的AI圈像高考放榜——所有人都盯着几个分数。GPT-4.1得分多少、Claude 4得分多少、Gemini多少。

今年没人比分数了。因为**大家走的方向不一样了**。

| 公司 | 方向 | 核心产品 |
|------|------|----------|
| OpenAI | 全能型 | GPT-5 + o3推理链 |
| Anthropic | Agent + 安全 | Claude 4 + Claude Code + MCP |
| Google | 长上下文 + 低价 | Gemini 2.5 Pro（百万token） |
| DeepSeek | 开源 + 性价比 | R1→R2 价格屠夫 |
| Meta | 整合生态 | Llama 4 + Manus（收购） |

这不是"谁比谁强"的问题了——是你**要什么就选什么**。

要写代码 → Anthropic
要大文档分析 → Google
要便宜 → DeepSeek
要全能 → OpenAI
要自家产品AI化 → Meta

## Agent从概念到标配

半年前我说"Agent是趋势"——是预测。

现在说"Agent是标配"——是描述。

我认识的所有程序员都在用某种 AI Agent 写代码。不是"试试看"，是**每天依赖**。

这半年的变化：

- **一月**：Agent还是极客玩具
- **三月**：Claude Code引爆编程Agent
- **五月**：行业共识——不用Agent写代码是浪费时间
- **六月**：Agent开始向非编程领域扩散（客服、运维、数据分析）

## 价格战打起来了

这是今年最戏剧性的变化。

DeepSeek-R2 的价格低到离谱——不到 GPT-5 的十分之一。逼得 OpenAI 和 Anthropic 都开始降价。

结果是什么？**用 AI 越来越便宜了。**

一年前用 GPT-4 API 写一个中型项目可能要几十美元。现在用 R2 几分钱。

这个趋势对创业公司是天大的利好——你可以用 AI 做很多以前做不起的事了。

## 最大的变化不在技术，在心态

去年这个时候，问一个普通人"你用过AI吗"，一半人说"没用过"。

今年再问，**几乎没人说没用过了**。不是"我偶尔用"，就是"我天天用"。

AI 从"尝鲜"变成了"基础设施"。就像你不会问别人"你用过手机吗"——因为默认大家都用。

## 下半年会怎样

我赌三件事：

1. **Agent 会进到更多非技术行业。** 会计、法律、HR、物流——不是替代人，是让人效率更高。
2. **MCP 生态会爆发。** 当所有软件都有 MCP 接口，Agent 能做的事就指数级增长。
3. **会出现"Agent时代"的第一个大的隐私丑闻。** 某个 Agent 泄露了大量用户数据，引发监管风暴。

第三条不是赌不赌的问题——是迟早的。
