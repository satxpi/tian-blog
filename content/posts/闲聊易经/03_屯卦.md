# 闲聊易经|03屯卦：刚开始，乱一点很正常

新买的耳机，第一次连蓝牙，总要折腾一会儿。

手机搜不到，说明书看不懂，按键长按还是短按也分不清。你一边怀疑耳机坏了，一边怀疑自己笨。

其实很多事刚开始都这样。

不是坏了。

是还没连上。

屯卦就是讲这个阶段。卦象名叫**水雷屯**，上面是坎水，下面是震雷。

它长这样：

```
上爻 ━ ━
五爻 ━━━
四爻 ━ ━
三爻 ━ ━
二爻 ━ ━
初爻 ━━━
```

雷在水下，想动，却被水困着。

像春天第一声雷还闷在云里，像种子刚破壳，泥土还压在头上。

屯，不是停死。

是初生的艰难。

卦辞说：**元亨利贞，勿用有攸往，利建侯。**

开始是有希望的，路也能通，但要守正。关键是“勿用有攸往”。刚开局，不要急着远行，不要一上来就铺大摊子。

“利建侯”，先立一个能管事的中心。

创业也好，学习也好，搬家也好，刚开始最怕什么都想做。屯卦说，先把秩序立起来。谁负责，先做哪一步，粮草在哪里，边界在哪里。

《象传》说：**云雷屯，君子以经纶。**

经纶，就是整理丝线。

乱线不能硬拽，越拽越打结。要一根一根理。

屯卦的六爻，也是一团乱线慢慢理开的过程。

**初九：磐桓，利居贞，利建侯。**

磐桓，是徘徊不前。

刚起步，走两步又停，想冲又不敢冲，很正常。

这时候最重要的不是速度，是位置。先居贞，守住正道；再建侯，确定主心骨。

一个团队刚成立，别急着喊口号。先定规则，定分工，定谁拍板。一个人刚改习惯，也别急着一天变完。先把闹钟、路线、工具准备好。

**六二：屯如邅如，乘马班如。匪寇婚媾，女子贞不字，十年乃字。**

这爻很有画面。

骑着马转来转去，像遇到阻拦。来的不是强盗，而是求婚的人。女子守正，不急着答应，十年才许嫁。

生活里很多机会也这样。

它看起来像麻烦，其实可能是合作；看起来像催促，其实需要确认。不要因为对方热情，就立刻答应。也不要因为局面乱，就把好意当坏意。

屯卦说，慢一点看清楚。真正合适的关系，经得起等待。

**六三：即鹿无虞，惟入于林中，君子几不如舍，往吝。**

想追鹿，却没有向导，只会钻进林子。

这像人一看到机会就上头。

朋友说某个行业赚钱，马上投钱；网上说某个方法有效，马上照抄；看到别人辞职自由，自己也想跟。

没有虞人，没有地图，没有判断，只凭兴奋追进去，最后多半迷路。

君子看见苗头，不如舍。放弃不是怂，是知道自己现在不具备条件。

**六四：乘马班如，求婚媾，往吉，无不利。**

还是马队徘徊。

但这一爻不同，方向清楚了。

去求婚，去建立关系，去促成合作，吉。

屯卦不是叫人永远等待。前面看不清要停，到了这里看清了就要去。

生活里也是。一个项目磨合够了，该签就签；一段关系确认真诚，该靠近就靠近；一个机会准备成熟，该出手就出手。

犹豫太久，也会错过。

**九五：屯其膏，小贞吉，大贞凶。**

膏，是滋润万物的油脂雨露。屯其膏，是恩泽被堵住，下不去。

位置在五，本来该发号施令、分配资源，却卡住了。

这时候小事守正还吉，大事硬撑就凶。

公司现金流紧张，就先保核心业务；家里预算紧，就先保必须开支；自己精力有限，就先做最重要的一件事。

局面堵的时候，别装大方，也别画大饼。小处稳住，比大处逞强好。

**上六：乘马班如，泣血涟如。**

到了最后还在原地打转，就不只是困难了。

是拖到伤心。

有些事一开始乱很正常，乱一年、两年、很多年，还不肯整理，就会变成消耗。关系不定，工作不定，方向不定，每天都像骑马绕圈，眼泪都快哭干。

屯卦最后很重。

它提醒我们，开局难可以理解，但不能把“难”当成永远不行动的借口。

屯卦讲新生。

新生不是电影里一束光照下来，万物自动展开。新生往往是泥、雨、雷声、误会、等待、试探，还有几次走错路。

所以刚开始乱一点，不丢人。

丢人的是乱了还不理。

把线头找出来，一根一根理。先立主心骨，再看清人和路。该舍的舍，该往的往。

春雷总会从水底滚出来。

但你得先把第一团乱线解开。
屯卦还有一个很实用的提醒：别把开头的混乱当成命运。

很多人刚开始做一件事，发现不顺，就马上给自己下结论：我不适合，我没天分，这条路不对。其实未必。种子刚出土时也歪歪扭扭，根还浅，叶还嫩，风一吹就晃。它需要的不是自我否定，而是继续扎根。

只要方向还正，主心骨还在，乱一点可以整理，慢一点可以推进。

以上均为个人观点，如有错误的地方，还请海涵。
