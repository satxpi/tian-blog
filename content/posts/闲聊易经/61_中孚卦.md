# 闲聊易经|61中孚卦：真心这东西，骗不了水面

有些话，嘴上说得漂亮，别人却不信。也有些人话不多，坐在那里，你就觉得稳。

中孚卦的卦象是**风泽中孚**。

它长这样：

```
上爻 ━━━
五爻 ━━━
四爻 ━ ━
三爻 ━ ━
二爻 ━━━
初爻 ━━━
```

中孚是风泽中孚。风在泽上，水面会动。心里的诚，也会在外面起纹路。豚鱼都能感应而吉，说明诚信深到粗微都能触及。

卦辞说：**豚鱼吉，利涉大川，利贞。**

卦辞里的关键，不是简单说好坏，而是把能走和不能乱走同时摆出来。中孚卦有它能通的地方，也有它必须守住的分寸。看懂这一层，才不至于把提醒听成口号。

《象传》说：**泽上有风，中孚；君子以议狱缓死。**

君子以议狱缓死。越是关乎生死的大事，越要凭诚心反复审议，缓一缓。真诚不是冲动判人，而是怕自己误伤。

中孚卦真正细的地方，在六爻。

**初九：虞吉，有他不燕。**

心有所安则吉；有别的杂念就不安。诚信的开始，是内心不乱。你嘴上说相信，心里另有盘算，水面马上起皱。

**九二：鸣鹤在阴，其子和之；我有好爵，吾与尔靡之。**

鹤在阴处鸣叫，小鹤应和。我有好酒，愿与你共享。真诚会回应真诚，不必站在聚光灯下。好东西愿意分享，信就深了。

**六三：得敌，或鼓或罢，或泣或歌。**

遇到对手，一会击鼓一会停，一会哭一会歌。内心被外物牵着走，就失去中孚。情绪太摇，别人很难信你。

**六四：月几望，马匹亡，无咎。**

月亮快圆，马匹失去，无咎。接近圆满时，舍掉私配偶也没问题。为了更大的信，放下一点私人绑定，反而清明。

**九五：有孚挛如，无咎。**

诚信紧紧相连，无咎。到了九五，信不是口头承诺，而是能把人连住。真正的组织、家庭、朋友，靠的就是这根看不见的绳。

**上九：翰音登于天，贞凶。**

鸡声飞上天，守着也凶。声音很高，不代表诚意很深。喊得越大，越可能空。中孚到最后警惕虚声。

中孚卦说，真心不是表演。

它会穿过话术，落到水面上。

你骗得了一时，骗不了纹路。
中孚最怕“我都说了你还不信”。

其实很多时候，别人不信，不是因为你说得不够多，而是因为你的水面不稳。

嘴上说没关系，脸已经冷了。

嘴上说为你好，手里全是控制。

嘴上说长期合作，算盘打得啪啪响。

人不是只听字。

人会听语气，看动作，记前后是否一致。

中孚的孚，像小鸟孵蛋。它不是喊出来的，是温度一天天守出来的。

所以真正的信任，很慢。

你答应的事做到了，一次。

难听的话你也诚实说了，一次。

有利益的时候你没有偷偷多拿，一次。

别人脆弱的时候你没有趁机压他，一次。

这些一次一次，才把信孵出来。

到那时，不必大声，水面自己会回应。
中孚也不是傻白甜。

真诚不等于谁的话都信。

恰恰因为心里有孚，才更怕冤枉人，也更怕被虚声骗。

所以《象传》说议狱缓死。

这四个字里有很深的克制。

面对一个错误，一个嫌疑，一个让你愤怒的人，先不要急着判死。再问一遍，再查一层，再听另一边的话。

很多关系里的误会，也是这样。

你看到一句消息没回，就判对方不在乎。

你听到一句转述，就判朋友背叛。

你抓住一个表情，就给整个人定罪。

这都不是中孚。

中孚的诚，是愿意接近真相，而不是急着维护自己的情绪。

真诚的人，心是热的。

但判断不能烫。
再往小处说，这一卦也不是挂在墙上的大道理。

它落在每天的细节里。

一顿饭怎么吃，一句话怎么回，一个念头起来以后要不要马上跟着走。

卦辞和爻辞看起来古老，其实说的都是这些瞬间。

人不是在大场面里才显出命运。

更多时候，是在没人鼓掌的地方，一点点把自己过成了现在的样子。
如果把它放回今天的生活，其实不用想得太玄。

我们每天都在这些卦里进进出出。

早上出门前的一个决定，饭桌上的一句忍住，工作里的一次复盘，关系里的一点诚意，都算。

《易经》最有意思的地方，不是把人带到云里，而是把人按回地上。

它让你看见，所谓吉凶，很多时候不是天突然伸手，而是你在某个位置上，做了合适或不合适的事。

位置对，分寸对，普通日子也会慢慢顺。

位置错，分寸乱，再大的热闹也会露出缝。
所以读这一卦，别急着问它能不能预测什么。

先问自己一句：我现在站在哪一爻。

是刚开始，还是已经过头；是该进，还是该停；是该把话说开，还是该把嘴闭上。

问清这个，比求一个神秘答案更有用。

以上均为个人观点，如有错误的地方，还请海涵。
