# 闲聊易经|62小过卦：小事上过一点，大事上别逞强

有时候，多说一句谢谢，多走两步送人，多检查一遍文件，看起来有点过。可这些小过，常常救命。

小过卦的卦象是**雷山小过**。

它长这样：

```
上爻 ━ ━
五爻 ━ ━
四爻 ━━━
三爻 ━━━
二爻 ━ ━
初爻 ━ ━
```

小过是雷山小过。雷在山上，声音过了山头，却不是大雷改天换地。卦辞说可小事，不可大事。飞鸟留下声音，不宜往上，宜往下。

卦辞说：**亨，利贞；可小事，不可大事。飞鸟遗之音，不宜上，宜下，大吉。**

卦辞里的关键，不是简单说好坏，而是把能走和不能乱走同时摆出来。小过卦有它能通的地方，也有它必须守住的分寸。看懂这一层，才不至于把提醒听成口号。

《象传》说：**山上有雷，小过；君子以行过乎恭，丧过乎哀，用过乎俭。**

君子在行为上过于恭敬，丧事上过于哀伤，用度上过于节俭。这里的过，是小处多一点分寸，不是大处乱来。

小过卦真正细的地方，在六爻。

**初六：飞鸟以凶。**

鸟一开始就往上飞，凶。小过最怕一上来就拔高。明明是小问题，非要上纲上线；明明能力未到，非要冲天。

**六二：过其祖，遇其妣；不及其君，遇其臣，无咎。**

越过祖父，遇到祖母；不到君王，遇到臣子。没到最高处，但遇到合适的人，无咎。小过贵在找恰当层级。

**九三：弗过防之，从或戕之，凶。**

不能稍微过度防备，跟随者可能伤你，凶。有些风险要提前多防一点。别嫌麻烦，门多锁一道，账多核一遍。

**九四：无咎，弗过遇之；往厉必戒，勿用永贞。**

无咎，但不要过度追求，只是遇到就好。往前有危险，必须警戒，不要固执长守。小过讲灵活，不讲死扛。

**六五：密云不雨，自我西郊；公弋取彼在穴。**

云很密却不下雨，从西郊来。公去射取穴中之物。气势有了，雨没下来，就做具体小事。别空等大局，先抓眼前洞里的东西。

**上六：弗遇过之，飞鸟离之，凶，是谓灾眚。**

没有遇合却飞过了头，鸟遭网罗，凶。这是灾。小过到极处，就是越界。小心过一点可以，飞太高就是找网。

小过卦像生活里的小心。

多一点礼，多一点哀，多一点省。

不壮观。

但很多平安，就是这些不壮观的小事换来的。
小过卦有一种很朴素的智慧：大处别乱，小处多做。

很多人刚好反过来。

大方向上随便赌，小细节上又懒得管。

人生选择拍脑袋，合同条款不细看；投资重仓凭感觉，止损规则没有；跟亲近的人说话很冲，却在外人面前拼命礼貌。

小过不是这样。

它说，大事不可乱过，小事可以稍微过一点。

该恭敬的时候，多弯一点腰。

该哀伤的时候，多给一点时间。

该节俭的时候，多省一点无谓开销。

这些小小的“过”，不是窝囊，是给生活留缓冲。

飞鸟遗音，不宜上，宜下。

鸟飞得太高，声音还在，人已经看不清了。

落低一点，贴近地面一点，反而大吉。

很多时候，我们不是不够优秀。

是太想证明自己高。
小过还教人一种低处的体面。

比如开会前早到五分钟。

比如答应别人的事，哪怕很小，也回个结果。

比如去别人家吃饭，随手把椅子推回去。

这些事太小，小到没人给你发奖状。

可人与人的信任，常常就从这些小地方长出来。

相反，一个人若在小事上总是差一点，时间久了，别人心里会有数。

他可能不坏。

但不稳。

小过不是让人卑微，而是让人在细节上多一点敬畏。

因为大事来临时，你临时装不出可靠。

可靠是小事养出来的。

飞鸟的声音落下来，人听见了，也记住了。

你平时怎么过小事，命运往往就怎么把大事交给你。
再往小处说，这一卦也不是挂在墙上的大道理。

它落在每天的细节里。

一顿饭怎么吃，一句话怎么回，一个念头起来以后要不要马上跟着走。

卦辞和爻辞看起来古老，其实说的都是这些瞬间。

人不是在大场面里才显出命运。

更多时候，是在没人鼓掌的地方，一点点把自己过成了现在的样子。
如果把它放回今天的生活，其实不用想得太玄。

我们每天都在这些卦里进进出出。

早上出门前的一个决定，饭桌上的一句忍住，工作里的一次复盘，关系里的一点诚意，都算。

《易经》最有意思的地方，不是把人带到云里，而是把人按回地上。

它让你看见，所谓吉凶，很多时候不是天突然伸手，而是你在某个位置上，做了合适或不合适的事。

位置对，分寸对，普通日子也会慢慢顺。

位置错，分寸乱，再大的热闹也会露出缝。

以上均为个人观点，如有错误的地方，还请海涵。
