## 因子是什么

量化里说的"因子"，本质就是一个数字，用来预测这只股票接下来会涨还是会跌。

最简单的例子：股票的市盈率（PE）。PE 低意味着这只股票可能被低估，历史数据显示低 PE 的股票长期表现往往好于高 PE 的股票。这就是一个有效的"价值因子"。

但光靠一个因子是不够的。量化的核心工作，是找到一批有预测能力的因子，组合起来。这就是"Alpha 因子挖掘"。

---

## 因子的分类

常见的因子大致分这几类：

### 技术因子

基于价格、成交量数据计算，来源是 K 线图：

- **动量因子**：过去 N 天的涨幅。动量效应在 A 股存在，但比美股弱
- **反转因子**：过去 1-5 天的反弹，短期反转在 A 股比较显著
- **波动率因子**：低波动率股票长期收益更好（低波动异象）
- **换手率因子**：换手率突然放大，往往预示短期走强

### 基本面因子

来自财务报表：

- **价值因子**：PE、PB、PS 等估值指标
- **成长因子**：营收增速、利润增速
- **质量因子**：ROE、资产负债率、毛利率

### 情绪因子

反映市场参与者的情绪：

- **分析师预期修正**：预期上调的股票后续表现更好
- **北向资金流入**：外资持续流入的股票有一定的跟踪价值
- **融资融券比例**：融券比例高意味着市场看空

### 另类数据因子

近年越来越多量化机构开始用：

- **卫星图像**：停车场车辆数量预测零售商销售额
- **招聘数据**：公司招聘量增加预示业务扩张
- **新闻情感**：用 NLP 分析财经新闻的正负面情绪

## A 股有效的因子

测了几十个因子之后，说一下在 A 股实际有效的（基于 2020-2025 年日线数据，CSI500 成分股）：

**有效的：**

| 因子 | IC 均值 | 衰减速度 | 说明 |
|------|---------|----------|------|
| 短期反转（1-5日） | 0.04～0.06 | 快（1周内失效） | A股情绪驱动强，短期反转显著 |
| 换手率异动 | 0.03～0.05 | 中 | 放量突破有效，但噪音多 |
| ROE（季报） | 0.02～0.04 | 慢 | 基本面因子持续时间长 |
| 分析师预期上调 | 0.05～0.08 | 中 | 机构预期修正信号强 |
| 低波动 | 0.02～0.03 | 慢 | 长期有效，短期意义不大 |

> IC（信息系数）= 因子值与未来收益的相关性，|IC| > 0.02 认为有预测价值

**看起来好但实际有问题的：**

- **PE 低估值**：A 股周期性强，低 PE 不一定是便宜，也可能是业绩下滑的预兆
- **动量因子**：美股动量效应显著，但 A 股惯性比较弱，反而短期反转更强
- **大数据舆情**：噪声很多，信号质量参差不齐，用了反而拉低组合表现

## 用机器学习挖因子

手动构造因子效率低。现在主流的做法是用基因编程（Genetic Programming）或者遗传算法，让机器自动搜索有效的因子表达式。

### WorldQuant 101 Alpha

WorldQuant 公开了 101 个 Alpha 因子公式，可以直接用：

```python
# Alpha#1: (rank(Ts_ArgMax(SignedPower(((returns < 0) ? stddev(returns, 20) : close), 2.), 5)) - 0.5)
# 用 Qlib 表达式实现：
alpha1 = "Rank(TsArgMax(If(Ref($ret,0)<0, Std($ret,20), $close)**2, 5)) - 0.5"
```

这 101 个因子在 A 股的有效性参差不齐，但作为 baseline 和特征工程的起点很好用。

### 用 LightGBM 组合因子

单个因子的预测能力有限，把多个因子组合起来效果更好：

```python
import lightgbm as lgb
import pandas as pd

# 特征矩阵：每只股票每天的因子值
X_train = factor_df[['反转1d', '反转5d', '换手率', 'ROE', '低波动', ...]]
y_train = future_return_df  # 未来5日收益

model = lgb.LGBMRegressor(
    n_estimators=500,
    learning_rate=0.05,
    max_depth=5,
    subsample=0.8,
)
model.fit(X_train, y_train, eval_set=[(X_valid, y_valid)])

# 特征重要性
importance = pd.Series(
    model.feature_importances_,
    index=X_train.columns
).sort_values(ascending=False)
print(importance.head(10))
```

跑出来你会发现：一般分析师预期修正、换手率异动这类情绪类因子排在前面，基本面因子靠后——符合 A 股散户情绪主导的特点。

## 常见陷阱

### 未来信息泄漏

最容易踩的坑。比如用"季报 ROE"做因子，但季报是季末之后才公布的，如果用季末当天的 ROE，实际上用了当时买入时还不知道的信息。

正确做法：**所有因子都必须用公告日滞后的数据**，而不是报告期的数据。

### 市值暴露过度

很多因子本质上是小市值因子的变体——A 股小市值长期跑赢大市值，导致暴露了小市值风险的因子看起来很好。

解决方法：在因子测试时，对行业和市值做中性化处理：

```python
# 市值中性化
from sklearn.linear_model import LinearRegression

reg = LinearRegression()
reg.fit(market_cap.values.reshape(-1,1), factor.values)
factor_neutralized = factor - reg.predict(market_cap.values.reshape(-1,1))
```

### 幸存者偏差

测试时只用现存的股票，退市的股票不在数据里。实际上退市股票往往是亏损最严重的，不包含会高估所有策略的收益。

Qlib 的 instruments 配置支持历史成分股，可以规避这个问题。

## 结论

挖因子没有捷径。有效的因子通常来自对市场机制的理解，而不是盲目测试。A 股的主要特点是：

1. **情绪驱动强**，短期反转因子比动量因子更有效
2. **机构行为可预测**，分析师预期修正是比较稳定的信号
3. **小市值效应显著**，做因子时一定要做市值中性化，不然测出来都是小市值效应
4. **基本面因子有效但慢**，适合做中低频策略，不适合做高换手

机器学习能帮你更快地筛选和组合因子，但不能替代对市场的理解。模型找出来的因子，还是要问"为什么这个信号有效"，不然很可能是过拟合。
