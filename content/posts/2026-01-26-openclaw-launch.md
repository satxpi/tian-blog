1月25日，星期六。

我正在改博客的 CSS（就是你现在看到的这个网站），随手打开 GitHub Trending，看到一个不认识的项目排在第一。

**OpenClaw**。

点进去一看——发布 12 小时，6000 个 Star。到第二天早上，突破了 9000。

GitHub 历史上从没有项目在发布 24 小时内拿到这么多星标。

## 它到底是什么

简单讲：**OpenClaw 是第一个开源的、模型无关的智能体操作系统。**

"模型无关"是关键词——你可以接 OpenAI、接 Anthropic、接 DeepSeek、接本地模型。Agent 框架不管你用什么大脑，它只负责"手脚"。

你可以把它装在自己电脑上、云服务器上、或者树莓派上。然后通过 Telegram、WhatsApp、微信、飞书这些 App，随时随地指挥它干活。

这就是"养虾"——你自己养一个 AI 助手，完全属于你。

## 怎么用

装好后，你在手机上的 Telegram 发条消息：

> "帮我把这个 PR 审查一下，写个 review comment"

它在你的电脑上：
1. 打开浏览器登录 GitHub
2. 找到对应的 Pull Request
3. 逐行审查代码
4. 写出 review
5. 贴到 PR 评论区
6. 回你一条消息："审查完了，提了 3 个建议，你看看。"

整个过程中你不用碰电脑。

## 跟 Manus/Devin 有什么不同

| | Manus/Devin | OpenClaw |
|---|-------------|----------|
| 开源 | 否 | MIT |
| 运行位置 | 云端 | 你的机器 |
| 接的模型 | 固定 | 任意 |
| 数据归属 | 服务商 | 你自己 |
| 扩展方式 | 等官方更新 | 社区插件 |
| 价格 | $500/月起 | 免费 |

本质区别就是一句话：**OpenClaw 是装你家里的发电机，Manus 是公共电网。**

## 为什么这么火

我猜三个原因：

**第一，时机。** 2026 年 1 月，DeepSeek-R1 刚刚证明"开源模型能打"，整个社区对"开源 AI 能做到什么"这件事信心爆棚。

**第二，开箱即用。** 以前做 Agent 要自己搭 LangChain，写几百行代码。OpenClaw 一条命令装完就能用。

**第三，中国人推起来的。** 消息平台支持（微信/飞书/Telegram）+ 腾讯云一键部署，让中国开发者成了 OpenClaw 最大用户群。中文"小龙虾"这个昵称，比英文 "Claw" 好听一百倍，传播极快。

## 但别高兴太早

我用了一天，就发现了问题：

- 配置复杂——虽然一条命令装完，但配置各种平台、密钥、权限，还是挺麻烦的
- 安全性——你给了它本地终端权限，它就能执行任何命令。万一模型被绕过了呢？
- 中文社区文档少——大部分文档、讨论都是英文的

不过这些问题不影响一个事实：

**OpenClaw 把 Agent 从大厂手里解放出来了。**

以前 Agent 是 Google、Microsoft、OpenAI 这些大公司的专属。现在你可以在自己电脑上跑一个，不用交费、不怕数据泄露、想怎么扩展就怎么扩展。

这件事的意义，怎么强调都不过分。
